# Atividade: Introdução à Engenharia de Dados 📊

Este repositório foi criado para o desenvolvimento do **[4.1.8. Exercício com Streamlit](https://raw.githubusercontent.com/monteiro74/aulas_2026_1/refs/heads/main/figuras/Exercicio2b.png)** da disciplina de Introdução à Engenharia de Dados. O objetivo da atividade foi realizar o deploy de uma aplicação Python utilizando o framework **Streamlit** integrado ao **GitHub**.

🚀 **Acesse o Dashboard online aqui:** 👉 [Dashboard de Vendas - Carla Eduarda](https://app-dashboard-carla-eduarda-a-izidorio.streamlit.app/)

---

## 📁 Arquivos da Atividade

* `streamlit_app3.py`: Script Python com a lógica e os gráficos do dashboard.
* `vendas_dashboard.csv`: Base de dados de vendas utilizada pela aplicação.
* `requirements.txt`: Lista de dependências (`pandas` e `plotly`) necessárias para o servidor rodar o código.

---

## 🛠️ Tecnologias Utilizadas

* **Python** (Pandas e Plotly Express)
* **Streamlit Community Cloud**
* **GitHub**
